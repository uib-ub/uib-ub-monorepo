# Exhibition Template (WIP)

The exhibition template is a template for creating exhibitions. It is a Next.js project that uses [Nextra](https://nextra.site/) for documentation, the [Next.js](https://nextjs.org/) framework and the [Tailwind CSS](https://tailwindcss.com/) library.

## Getting Started

To get started, clone the repository and install the dependencies:

```bash
git clone https://github.com/uib-ub/exhibition-template.git
```
