# Apps

The University of Bergen Library have developed a number of web applications and services for the public.

<Cards num={2}>
  <Cards.Card title="Stadnamnportalen" href="./apps/stadnamnportalen">
    <div className="p-4">
      _Stadnamnportalen_ is a web application that allows you to search for information about the names of places in Norway.
    </div>
  </Cards.Card>
  <Cards.Card title="Decriminalizing History" href="./apps/decriminalizing-history">
    <div className="p-4">
      _Decriminalizing History_ is an exhibition that is a celebration of diversity but also a declaration of queer independence, making visible rights and fights that still affects us.
    </div>
  </Cards.Card>
  <Cards.Card title="Find SDG research" href="./apps/find-sdg-research">
    <div className="p-4">
      _Find SDG research_ is a web application that allows you to search for research about the Sustainable Development Goals (SDGs).
    </div>
  </Cards.Card>
  <Cards.Card title="Landslovjubileet" href="./apps/landslovjubileet">
    <div className="p-4">
      _Landslovjubileet_ is an exhibition about the 750th anniversary of the Laws of the Land of Norway in 2024. This exhibition is the main event in the celebration of the Laws of the Land jubilee in Vestland County.
    </div>
  </Cards.Card>
  <Cards.Card title="Nansen og Bergenserne" href="./apps/nansen-og-bergenserne">
    <div className="p-4">
      _Nansen og Bergenserne_ is an exhibition that is a celebration of the 100th anniversary of the Norwegian Constitution.
    </div>
  </Cards.Card>
  <Cards.Card title="Sok og Skriv" href="./apps/sok-og-skriv">
    <div className="p-4">
      _Sok og Skriv_ is a web application that helps students with information gathering and academic writing.
    </div>
  </Cards.Card>
  <Cards.Card title="Exhibition Template" href="./apps/exhibition-template">
    <div className="p-4">
      _Exhibition Template_ is a template for creating exhibitions. It is a Next.js project that uses [Nextra](https://nextra.site/) for documentation, the [Next.js](https://nextjs.org/) framework and the [Tailwind CSS](https://tailwindcss.com/) library. It is a work in progress and will be updated as we develop the template.
    </div>
  </Cards.Card>
</Cards>
