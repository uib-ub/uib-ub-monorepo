# Toolkit

This is a list of tools and libraries that can be used to build applications in the cultural heritage and language domains.

## IIIF vizualisation

* [@samvera-labs/clover-iiif](https://github.com/samvera-labs/clover-iiif) - A IIIF viewer for Manifest images and metadata.
* [openseadragon](https://openseadragon.github.io/) - OpenSeadragon is an open-source, web-based viewer for high-resolution zoomable images, implemented in pure JavaScript.
* [yith](https://github.com/mathewjordan/yith) - Yith is a IIIF manifest storytelling library.

## Data manipulation

* [json-ld](https://json-ld.org/) - JSON-LD is a W3C Recommendation for embedding Linked Data in HTML and other formats. It provides a way to use the full power of Linked Data in Web pages and other formats in a way that is easy for authors and equally easy for developers to parse and generate.

## Maps

* [OpenFreemap](https://openfreemap.org/) - OpenFreemap is a free and open-source map server that allows you to create and share maps with your friends and family.
* [react-map-gl](https://visgl.github.io/react-map-gl/) - React Mapbox GL JS is a React library for Mapbox GL JS. [Github](https://github.com/visgl/react-map-gl)

## Deduplication

* [czkawka](https://github.com/qarmin/czkawka) - Tool for finding and removing duplicate files, empty folders, similar images and similar files.
