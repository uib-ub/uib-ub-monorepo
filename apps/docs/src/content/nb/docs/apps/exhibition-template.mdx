# Exhibition Template (WIP)

Utstillingsmalen er en mal for å opprette utstillinger. Det er et Next.js-prosjekt som bruker [Nextra](https://nextra.site/) for dokumentasjon, [Next.js](https://nextjs.org/)-rammeverket og [Tailwind CSS](https://tailwindcss.com/)-biblioteket.

## Komme i gang

For å komme i gang, klon repositoryet og installer avhengighetene:

```bash
git clone https://github.com/uib-ub/exhibition-template.git
```
