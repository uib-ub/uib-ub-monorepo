# Avstemming av data

UB-SPES har brukt vokabularer/konseptskjemaer/registre fra starten av. Fra kortkataloger til kontrollerte strenger og SKOS-konsepter. Dataene er tilgjengelige i [UB-SPES SPARQL endpoint](https://sparql.ub.uib.no/sparql) og konseptskjemaene er tilgjengelige på [marcus.uib.no/conceptschemes](https://marcus.uib.no/conceptschemes). "Løvetann emneregister" er også inkludert.

## Arbeidsflyt

*NB!:* JSON-representasjon av registre er ikke tilgjengelig for øyeblikket!

<Steps>
### Last ned dataene
 
Last ned ConceptScheme som JSON fra denne listen
 
### Avstem
 
Importer JSON til OpenRefine eller annet passende verktøy som PoolParty eller VocBench. Kulturnav støtter ikke Reconciliation Service API v0.2, så det kan ikke brukes direkte med OpenRefine. _En løsning bør finnes for dette._

### Eksporter

Eksporter de avstemte dataene som RDF/XML eller Turtle og last opp til [Marcus SPARQL endpoint](https://sparql.ub.uib.no/sparql)

</Steps>
