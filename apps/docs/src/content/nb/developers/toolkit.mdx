# Verktøykasse

Dette er en liste over verktøy og biblioteker som kan brukes til å bygge applikasjoner innen kulturarv- og språkdomener.

## IIIF-visualisering

* [@samvera-labs/clover-iiif](https://github.com/samvera-labs/clover-iiif) - En IIIF-viser for Manifest-bilder og metadata.
* [openseadragon](https://openseadragon.github.io/) - OpenSeadragon er en open-source, webbasert viser for høyoppløselige zoombare bilder, implementert i ren JavaScript.
* [yith](https://github.com/mathewjordan/yith) - Yith er et IIIF manifest storytelling-bibliotek.

## Datamanipulering

* [json-ld](https://json-ld.org/) - JSON-LD er en W3C-anbefaling for å bygge inn Linked Data i HTML og andre formater. Den gir en måte å bruke den fulle kraften i Linked Data på nettsider og andre formater på en måte som er enkel for forfattere og like enkel for utviklere å parse og generere.

## Kart

* [OpenFreemap](https://openfreemap.org/) - OpenFreemap er en gratis og open-source kartserver som lar deg opprette og dele kart med venner og familie.
* [react-map-gl](https://visgl.github.io/react-map-gl/) - React Mapbox GL JS er et React-bibliotek for Mapbox GL JS. [Github](https://github.com/visgl/react-map-gl)

## Deduplisering

* [czkawka](https://github.com/qarmin/czkawka) - Verktøy for å finne og fjerne duplikatfiler, tomme mapper, lignende bilder og lignende filer.
