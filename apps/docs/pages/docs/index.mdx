# Introduction

The University of Bergen Library is using this monorepo for web applications and services. The monorepo is managed by [Turborepo](https://turborepo.com/). 

## Setup

```sh
git clone git@github.com:uib-ub/uib-ub-monorepo.git
cd uib-ub-monorepo
```

### Build

To build all apps and packages, run the following command:

```sh
npm run build
```

### Develop

To develop all apps and packages, run the following command:

```sh
npm run dev
```
