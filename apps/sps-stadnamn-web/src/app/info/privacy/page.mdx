# Personvern
Stadnamnportalen benytter tjenesten Plausible til å registrere bruksstatisitkk, eksempelvis antall besøkende, land, nettleser og operativsystem. Plausible benytter ikke informasjonskapsler, og ingen personlig informasjon blir lagret. [Les mer om Plausible](https://plausible.io/data-policy)
