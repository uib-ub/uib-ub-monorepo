# Bruksanvisning
Stadnamnportalen har et [overordnet søk](/view/search) og [søkevisninger for enkelte datasett](/datasets). I førstnevnte skal vi på sikt slå sammen alle treff som er tildelt en stadnamn-ID i språksamlingenes
database (SNID). I datasettvisningene vil man kunne bruke søkefiltre som er spesifikke for hvert datasett.
