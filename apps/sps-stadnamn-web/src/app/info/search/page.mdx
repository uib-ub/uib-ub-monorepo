# Om Stadnamnsøk
Eit oppslag i stadnamnsøket er eit eige uttrykk eller konsept forskjellig frå alle andre stadnamnoppslag i det området namnet er belagt. Då eit stadnamn kan brukast både om fleire lokalitetar og om forskjellige typar lokalitetar, samlar eit stadnamnuttrykk same namn i ei oppslag i portalen.
Det pågår stadig eit arbeid med å få plassert kjeldeformer til rett språklig uttrykk, og innan ein stadnamnform er blitt lagt til eit stadnamnoppslag, vil ein kjeldeform stå som eiga oppslag så det går an å søkje fram alle kjeldeformene i Stadnamnportalen.
